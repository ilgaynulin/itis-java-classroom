Функции/Процедура

типВозвращаемогоЗначение названиеФункции(типАргумента1 имяАргумента1, типАргумента2 имяАргумента1 ...)

// объявили свою функцию возведения в квадрат
static int squared(int number) {
	return number * number;
}

типВозвращаемогоЗначение может быть:
void - функция ничего не возвращает
int, boolean и т.д. - возвращает значение с соответствующим типом

Функция если возвращает значение. Процедура, если ничего не возвращает.

Передача аргументов в функцию/процедуру:

С ссылочными типами (массив)
В функцию передаётся только ссылка. Все изменения в памяти произведённые через ссылку будут отображаться в переданной переменной

С примитивными типами данных - в функцию переменная передаётся по значению. Т.е. создаётся копия переменной. Т.е. изменения переменной внутри функции не отражается на той, с которой её вызвали.
