Одномерный массив

Объявление
типЭлементов идентификаторМассива[];
Пример:
int array[] = {1, 4, 7}; // объявление и инициализация
double[] measurements = new double[10]; // Объявление массива с определённым размером

Индексация в массивах в Java начинается с 0

Получение значения элемента массива
measurements[0]; // Получение первого элемента из массива
measurements[2]; // получение 3-го элемента из массива

типЭлементов идентификаторМассива[][];
