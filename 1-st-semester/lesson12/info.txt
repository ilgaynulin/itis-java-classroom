Абстрактный класс

Объявление с помощью ключевого слова abstract

Пример 
public abstract class Figure {
	
}

Можно унаследоваться от абстрактного класса, но при этом необходимо реализовать все его абстрактные методы, либо обозначить сам класс абстрактным

Невозможно создать объект от абстрактного класса

Абстрактный класс может иметь реализованные методы и абстрактные методы

Абстрактный класс может описывать состояние объекта 

Объекты реализующие абстрактный класс можно присвоить к переменной-ссылки абстрактного класса

Пример:
Rectangle rectangle = new Rectangle(2.0, 3.0);
Square square = new Square(3.3);
Figure[] figs = new Figure[2];

figs[0] = square;
figs[1] = rectangle;
