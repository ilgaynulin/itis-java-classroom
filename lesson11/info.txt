Object - класс который является наследником для всех классов в java

- boolean equals(Object object) - сравнивает объект из аргумента с самим собой. true - если объекты равны, false - если нет

- String toString() - строковое представление класса

- Class getClass() - получаем информацию о классе.

- int hashCode() - возвращает хэшкод объекта

- clone() - возвращает клон объекта.
